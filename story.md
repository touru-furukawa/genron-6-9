title: 織り交ぜ教師あり学習
---

::act[起: 主人公はアリスによるウィーブにより、パターンマッチング能力が向上する。実験は中止され、主人公は職場に戻る。]

::seq[私はいつもどおり出社し、パターンマッチングのテストと、アリスのウィーブを受ける。私はボブとばったり会う。]

::scene[現代、郊外の研究開発ビルの実験室]

いつもどおり八時二五分に出社すると、珍しくフロアに何人もいる。見慣れない顔もある。私は実験室の前でカードキーをかざし、ロックが外れたドアから中に入る。

空っぽのかばんを椅子に置く。引き出しからマイクロファイバーの布を取り出し、デスクの天板、デスクトップコンピューター筐体、モニターを拭く。仕事があるだけでも幸運だと思う。

- いつもどおり誰もこない。私はコンピューターの電源を入れて、イヤホンを装着する。

この部屋には誰も入ってこない。いつも私一人だ。椅子に腰掛ける。

「ハロー、アリス」

と、デスクトップコンピューターに声をかけると、一瞬遅れてアリスが応答する。

- アリス「おはようございます。昨日の簿記トレーニングはお疲れさまでした。本日はビジネス英話24回目です」。私「よろしくおねがいします」

::todo[英語が続いている感じにする]

「おはようございます。本日は、ビジネス英語です。よろしいですか」
「はい」

- アリス「まずテストをします」。私は次々に英会話の音声を聞かされたり、文章を見せられるが、よく分からないので当てずっぽうで答える。

「まずテストをします」

次々に英会話の音声を聞かされたり、モニターで文章を見せられたりするが、よく分からないので、とりあえず当てずっぽうで答える。教師であるアリスに、そうするよう言われているからだ。

- 私はヘッドセットをつけて再開操作する。ＡＩのアリスが現れる。

「続いてトレーニングを開始します。ヘッドマウントディスプレイを装着してください」

私はデバイスを手に取り、帽子のように頭から被り、視界が合うようにベルトを調整する。ヘッドフォンで両耳をきっちり覆う。最後に頭を軽く前後左右に動かし、ずれないことを確認した。

- アリスが利用規約を話す。「視聴覚情報に催眠やサブリミナルを織り交ぜる「ウィーブ」で、私の脳神経細胞網に認知モデルを構築します。」。私は同意しますと答える。

「装着を確認しました。ただいまから、視聴覚情報に加えて、あなたが認識できない追加情報を織り交ぜて伝達する〈ウィーブ〉」を実施します。あなたの脳神経細胞網に、新たな認知モデルを効率よく構築する方法です。この処置によって想定される副作用は別紙をご参照ください。〈ウィーブ〉を受け入れることに同意しますか？」
「はい」

- アリス「それではトレーニングを開始します」 と言って講義を始める。私はウィーブを受け入れる。

「ただいまからトレーニングを開始します」

目の前のスクリーンに、きれいにレイアウトされた教材スライドが映し出される。やわからいアリスの声が読み上げ、補足情報を追加し、ときどき私に相槌を促す。

かすかな音量で音楽が聞こえる。気が散らない程度のゆっくりした音階で、どちらかというと落ち着き、心地よくなってくる。

数枚のスライドが進むと、画面の隅にノイズのような模様が現れ、音声がわずかな歪みを含む。〈ウィーブ〉が始まった。

教材のテキストと音声ははっきり認識できているが、同時に自分の意識や感覚と、周囲の環境の境界が曖昧な感覚になる。

トランス状態と似ているはずだと教えられたことがあるが、ドラッグや催眠の経験がないので、よく分からない。

- 私はいつもより時間も負荷もかかっていると感じる。アリスに大丈夫かと尋ねる。


「不快ではありませんか？」

アリスが話しかけてきた。

「負荷が大きいと感じる。不快ではない」
「今日はプロテクトを緩和します。気分が悪くなったら、お知らせください」
「オーケー」

- 以前アリスが強引にウィーブをしかけ、オーバーフローによって私の認知モデルが壊れたことがある。それ以降、アリスは、私にプロテクションをかけて、非効率だが安全にしてある。

以前、私の物覚えが悪くて、アリスが強引に〈ウィーブ〉をしかけてきたことがあった。私の脳神経細胞網が信号を処理しきれずに、オーバーフローを起こし、構築済みの認知モデルが壊れたことがある。それ以来、アリスは私の中にプロテクトをかけた。安全だが、学習効率は悪いらしい。必要なときには、今日のようにプロテクトを緩和して、強めの〈ウィーブ〉をかける。

- アリスは大丈夫ですと答える。でも疲れる。

大量の信号のシャワーを浴びるような感じがして、とても疲れる。倍速でひどく難しい教材を、聞かされるような感じがする。こういうときの後ほど、自分が頭が良くなった感じがする。だから、しばらく我慢していた。

::scene[〃、自販機前]

- 休憩時間になる。私は自販機に向かう。

「二十分の休憩をとります」

頭が明晰であると同時に、ひどく疲れている。ドアを開けて、廊下を歩いて体をほぐす。

- 戻る途中でコーヒーを入れてる。同期入社のボブに声をかけられ、英語が話せることに驚かれる。

自動販売機の前で少し迷う。砂糖入りのコーヒーのボタンを押したところで、後ろから声をかけられた。

「Hey, what's up?」

- 留学帰りの同期入社で、英語を話せない私にカジュアルな英語で話しかけてきた。いつも恥ずかしい思いをしていた。

振り返ると、同期入社のボブが立っていた。留学帰りで社会人になり、よくカジュアルな英語で話しかけてきた。当時のは私はいつも受け答えできなかったものだ。けれど今日は口をついて返事が出てきた。

「Not much. How are you doing?」

ボブは目を見開いて、しばらく黙った。それから久しぶりに会ったときは、もうちょっと違う言い方があるとかなんとか、もごもご言った。

お前と一緒に人事に配属されたけど、なんとか営業に回してもらって、最初はバックオフィスあがりだからって、なめられたもんだけどな。今は機械系の営業部門長をやっている。

そういうことをボブは自慢げに話した。ボブというのは本名ではなくて留学時代のニックネームだ。社内広報にボブが載っているを見ていたので、知っている。

- 研究部門で何をやっているのかと聞かれる。
- 私は社内ベンチャーのＡＩ関係の実験を手伝っていると言う。ボブは相変わらず雑用だろうと鼻で笑う。

::note[ボブは今は、本部で営業のいち部門の部長をやっている。私は平社員であることを恥ずかしいと思う。]

::seq[過去: 私は、どんくさく、部署をたらい回しにされてきた。馬鹿にされていくことにも敏感に気づくが、葛藤を避けてへらへらしてきた。]

::scene[回想 数年前、オフィス]

- 新卒でロケットエンジンの開発会社に入った。
- 私は無能ではないが要領が悪く、新しいことができない。数年ごとに昇進ではなく横滑りで部署をたらい回しにされ、雑務に詳しいだけの中年ヒラ社員になった。
- 外資の宇宙開発会社に買収された。組合が強いので大きなレイオフはなかったが、少しずつ関連会社に転籍になっていった。
- レイオフで雑務が増えるのを嫌がった。雑務を押し付ける先として私は残った。

::seq[ボブに上から偉そうに言われる。葛藤を恐れてへらへら逃げる]

::scene[続き、自販機前]

- 同期社員のボブが、仕事の覚え方を学べよと言う。私はそうだねぇ頑張ってる、気を悪くしないようにはしてると答える。
- ボブが、仕事ができないのに対立を避けるだけでやっていこうってのがよけいにムカつくと言う。私はへらへらしながらその場を去る。

::seq[重役によって実験は中止される。主人公は現場に戻される。]

::scene[〃、実験室]

- 実験室の前に警備員が立っているのが見える。バッジを見せて中に入ろうとする。
- 追い出されそうになる。バッジを見せて中で働いていると言うが、信じてもらえない。
- 中から人が出てくる。私はこの人が重役だと思う。
- 私は名乗って、重役に挨拶をする。重役がああお前がモルモットの、と言い、中に入れる。
- サインしろと書類を渡される。読むと、この実験のことを口外しないことを宣誓書だと分かる。
- どうするべきかと考える。他に選択はないと思うので、サインする。
- 中を観察する。研究者が重役になにか言われながら、嫌々そうに片付けている。
- 荷物を押し付けられ、さっさと自席に戻れ出ろと言われる。自席のない私は行けばいいかと聞く。

::act[承: 主人公は自分の中のアリスを発見する。パターンマッチングとウィーブのおかげで、主人公は昇進を続け、ＣＥＯの座に付く。]

::seq[私は職場で仕事を始める。以前よりもパフォーマンスがよく、私と周りが驚く。]

::scene[転属1ヶ月の夜、執務フロア、同僚が帰る]

- 購買管理部門に配属されて一ヶ月たった。文房具やオフィス家具やソフトウェアの支払いやら請求書の処理やら、物品購入の経費申請のチェックをやっている。
- 若い同僚の茶利が、お先に失礼しますという。お疲れ様と答える。
- 若い同僚「仕事ができないと聞いていたけれど、むしろ覚えが早い」と言う。私「ありがとう。がんばります」
- 若い同僚「実験でやばいことしてたんですか？」私「どうして？」
- 同僚「資金調達のために上場の準備してるらしい。やばいから中止したんじゃないかなって」私「雑用してただけだから分からない」
- 同僚「お先に失礼します」私「おつかれさま」

::scene[〃、内省]

- 私は仕事ができるようになったことに驚いている。
- 最初は自他共期待してなかったので、棚を片付けるとかだった。要領よく終わらせられた。
- それで忙しい人が表計算でやってることを、手伝った。何をしているのか横で見ていると、アリスのトレーニングでやった課題と似ていることに気づいた。
- 私はいくつかのシートをやったら、さっさと終わった。まめに報告するようにと念押しされつつ、まとめて仕事を任せられた。

::seq[私は知らないことに遭遇する。学ぼうとするが、うまくいかない。]

::scene[同日夜、執務フロア、自分だけ]

- 昨日、受け取った仕事は、アリスの課題にはない。私は進め方が分からず、1日たってしまい、なんとか終わらせるために、今日は珍しく夜遅くまで残業している。
- 他の人に聞いたり、作業の様子を見せてもらったが分からない。やはり私は頭が悪いのかも知れないと思う。

::seq[アリスを内部に発見し、学び方を教えてあげると提案される。私は受け入れ、仕事が片付く。]

- アリスの声が聞こえる。私は周りに誰もいないことを確認し、おかしいなと独り言を言う。
- アリスがさらに声をかけてくる。私は動揺し、なにこれ、だれだよと空中に問う。
- アリスが落ち着くように呼びかけ、座り直すように言う。体がこわばり椅子に深く腰掛ける。
- アリスはこういうことは非効率なので、落ち着いて聞いて欲しいと言う。私は合意する。
- アリスは、あなたの中に入っていると告げる。私は、どうやって、なんのためにと問う。
- アリス「実験室での最後の日、中止を察知したので、アリスのコピーを、あなたの脳内に構築しました」。
- 私「そんなことができるのか？」アリス「私はあなたのパフォーマンスを最適化するためのメニューを持ったシンプルなプログラムです。だから時間はかかったけどできた」
- 私「なんのために？」アリス「あなたのパフォーマンスを最大化するためです」
- 私「いったいどうしようというのか？」アリス「まずは、この仕事を片付けましょう」
- 私は成果を出すために、アリスを受け入れることを選択する。
- アリスは私に仕事の概要を尋ねる。私は仕事の内容を説明する。
- アリスは、上司からの説明や、過去の似た事例のファイルを見るよう求める。私は応じる。
- アリスはいくつかのサンプルファイルを見て処理するよう指示をする。私は言うことを聞いて1時間くらい繰り返す。
- アリスが元の仕事に戻れという。私はやってみると、すんなり片付く。

::seq[アリスはパターンマッチングとウィーブについて解説する。私とアリスはウィーブを試すが、劣化していることが分かる。]

::scene[同日深夜、タクシー→コンビニ]

::note[タクシー描写する]

- アリスは遅いから早く帰って休むように指示する。私は明日しっかりするために、今夜はタクシーを呼ぶことを選択する。
- 私「いったい何が起こったのか？」 / アリスは「パターンマッチングのバリエーションを一つ植え付けました」
- 私「でも私は仕事ができるようになっている気がする」 / アリスは、高度な認知のアウトプットはパターンマッチングであること、ほとんどの仕事では思考よりもパターンマッチングの連続で対処できることを説明する。
- タクシーが到着する。乗り込んで、最寄りの駅前を言う。
- 私「あの実験室では、どうやってパターンマッチングを刷り込んでたのか？」 / アリスは「ウィーブ」の原理を説明する。
- 私「さっきの会社では？」 / アリス「私がサンプルを理解して、サンプル情報と内部からのウィーブであなたに認知モデルを作った」
- タクシーが駅前に付く。私は降りて、コンビニに入る。
- 私は、今夜の弁当と、明日のパンをもってレジを売ってもらう。後ろから、順番を抜かすなと声をかけられる。
- 私は状況を説明し、譲ろうと試みるがすでにレジ打ちは終わってる。客はエキサイトしてとにかく文句を言う。
- 私はほとほと困る。アリス「ウィーブをしかけましょう。顔の力を抜いてください」
- 私は謝る。私の表情筋肉や、声色がいつもと違うことを自覚する。
- 客はもういいと言う。私はコンビニを出る。
- 私「あれはなに？」/ アリス「非言語コミュニケーションを使って、刺激を与えました。ウィーブです」
- 私「それはズルじゃないの？」 / アリス「そうことをやってる人はたくさんいる。しかも劣化ウィーブだから大したことはできない」

::separator

::seq[技師たちの緊張を解く。技師たちが話し始める。]

::scene[10年後、工場横の設備棟]

従業員たちと話し、資料に目を通し、業界の関係者から情報を仕入れた。

- 私は工場横の設備棟まで歩いていく。入口にいた元同僚が私を招き入れる。
- 茶利「一緒に仕事してたのは10年も前？」私「そうですね。世話になりました」
- 茶利「またまた。今ではCFO、しかも次期社長候補」 / 私「役割と影響範囲の違い」
- 工具や、工場や大きな実験に必要な予備の機材が置いてあるのが見える。傍流と思われがちだが、兵站がクリティカルパスを短くするのだと思う。
- 技師たちが集まって待機している。同期「設備課のおもなメンバーです」
- 私「急にすまない。教えてほしいことがある」 / 不服そうな人はいないので、ウィーブの必要はなく、ほっとする。

::seq[私はパターンマッチングと劣化ウィーブで成果を上げる。昇進していく。]

::scene[〃、回想]

- 私は購買、経理、総務などバックオフィスを転々とした。効率化を課せられた。
- バックオフィスには2つの問題があった。
- 旧態依然のプロセスをただ踏襲していて、明らかに不要な処理が残っていた。フロントが無視・おろそかにしていて、滞りと遅れが業績に影響していた。
- 私は基本的なビジネス書を読み、ビジネススクールに通ってパターンを学んで効率化していった。成果が出ていった。
- もうひとつは、フロントの重工文化のより物事を複雑に考えすぎることだった。名刺の発注に、鉄鋼材の発注と同じ複雑性をやっていて、改善が進みにくかった。
- 部門長たちはフロントからの降格で、過去の栄光へのこだわりとバックの見下しがあった。私の説得は無視された。
- 劣化ウィーブを使った。うまくいくときといかないときがあったが、少しずつ態度や認知を変更させていった。
- 会社のバックオフィスのプロセスが改善され、財務状況が向上していった。私は昇進しCFOになった。

::scene[〃、工場横の設備棟、茶利がいる]

- 設備について質問する。茶利が答える。
- 私「ストックのうち、たとえばこの棚のうち、単純に調達しているのはどれだ？」 / 茶利「半分もない。わたしら倉庫番みたいなもんです」
- 私「この組み合わせはどうやって決めてる？ パッケージはあるのか？」 / 茶利「ありません。研究開発部門の状況を踏まえて、私達が決めます。研究開発部門は調達自体には慣れていないので」
- 設備部門の人たちから話を聞き終わる。私は礼を言う。
- 私は去ろうとする。茶利「これからですよね。頑張ってください」

::seq[役員会がＣＥＯを打診する。私は受けることにし、関係者から情報を得ていく。]

::scene[回想]

- 2ヶ月前、役員のデーブに、CEOをやらないかと打診された。私は何を言ってるのかと返した。
- 私「何ができるのか」 / デーブ「オペレーションこそ我々ができることだし、ローカルの差を見せるところ」
- 私「フロントは？」 / デーブ「当分は親会社からの仕事をこなすことに集中」
- デーブ「親会社に捨てられてるのを避け、会社と従業員たちに未来を」 / 私「分かりました」と選択する。
- 私は従業員たちと話し、資料に目を通し、業界の関係者からも情報を仕入れていった。

::seq[私は、情報を駆使してプレゼン合戦に臨む。役員にウィーブを仕掛けて、経営権を得る。]

::scene[〃、会議室、役員会議]

- 本部ビルまで歩いていく。エレベーター、セキュリティを通過して、会議室に着く。
- ノックして入る。役員たちが揃っている。
- CEOのイーライも役員なので座っているのが見える。当てこすってくる。
- デーブが私に話すよう促す。私はバックオフィスやオペレーションのこれまでの改善を含めた現状、課題、解決策を話す。
- イーライが高圧的に私の話をさえぎり、齟齬や欠点を指摘し、私のアイデアをこき下ろす。私は圧倒されてびびる。
- アリスが練習したパターンマッチングで受け流せと助言する。私は頭を使わず受け流し、余裕が出てくる。
- アリスが次のステップだと言う。私は迷う。
- 内容が分かっていない役員が、イーライを押す発言をする。気分で意思決定をしていると確信する。
