---
title: 織り交ぜ教師あり学習
---

::act[起: 主人公はアリスによるウィーブにより、パターンマッチング能力が向上する。実験は中止され、主人公は職場に戻る。]

::seq[私はいつもどおり出社し、パターンマッチングのテストと、アリスのウィーブを受ける。私は同期社員とばったり会う。]

::scene[現代、郊外の研究開発ビルの実験室]

- 私が研究室のドアにカードキーをかざすとアンロックされる。私は中に入る。
- かばんを置いてマイクロファイバーの布でデスクやキーボードを掃除する。仕事に感謝をしている。
- いつもどおり誰もこない。私はコンピューターの電源を入れていく。
- アリス「おはようございます。本日は決算書トレーニング 24回目です」。私「よろしくおねがいします」
- アリス「まずテストをします」。私は次々に決算書を見せられるが、何が悪いのかよく分からず、ほとんど当てずっぽうで答える。
- 私はヘッドセットをつけて再開操作する。ＡＩのアリスが現れる。
- アリスが利用規約を話す。「視聴覚情報に催眠やサブリミナルを織り交ぜる「ウィーブ」で、私の脳神経細胞網に認知モデルを構築します。」。私は同意しますと答える。
- アリス「それではトレーニングを開始します」 と言って講義を始める。私はウィーブを受け入れる。
- 私はいつもより時間も負荷もかかっていると感じる。とても疲れる。

::scene[〃、自販機前]

::todo[同期社員の名前は莫武（ボブ）]

- 休憩時間になる。私は自販機に向かう。
- 戻る途中でコーヒーを入れてる。同期に声をかけられ、研究部門で何をやっているのかと聞かれる。
- 私は社内ベンチャーのＡＩ関係の実験を手伝っていると言う。同期社員は相変わらず雑用だろうと鼻で笑う。

::note[同期社員は今は、本部で営業のいち部門の部長をやっている。私は平社員であることを恥ずかしいと思う。]

::seq[過去: 私は、どんくさく、部署をたらい回しにされてきた。馬鹿にされていくことにも敏感に気づくが、葛藤を避けてへらへらしてきた。]

::scene[回想 数年前、オフィス]

- 新卒でロケットエンジンの開発会社に入った。
- 私は無能ではないが要領が悪く、新しいことができない。数年ごとに昇進ではなく横滑りで部署をたらい回しにされ、雑務に詳しいだけの中年ヒラ社員になった。
- 外資の宇宙開発会社に買収された。組合が強いので大きなレイオフはなかったが、少しずつ関連会社に転籍になっていった。
- レイオフで雑務が増えるのを嫌がった。雑務を押し付ける先として私は残った。

::seq[同期社員に上から偉そうに言われる。葛藤を恐れてへらへら逃げる]

::scene[続き、自販機前]

- 同期社員が、仕事の覚え方を学べよと言う。私はそうだねぇ頑張ってる、気を悪くしないようにはしてると答える。
- 同期社員が、仕事ができないのに対立を避けるだけでやっていこうってのが返ってムカつくと言う。私はへらへらしながらその場を去る。

::seq[重役によって実験は中止される。主人公は現場に戻される。]

::scene[〃、実験室]

- 実験室の前に警備員が立っているのが見える。バッジを見せて中に入ろうとする。
- 追い出されそうになる。バッジを見せて中で働いていると言うが、信じてもらえない。
- 中から人が出てくる。私はこの人が重役だと思う。
- 私は名乗って、重役に挨拶をする。重役がああお前がモルモットの、と言い、中に入れる。
- サインしろと書類を渡される。読むと、この実験のことを口外しないことを宣誓書だと分かる。
- どうするべきかと考える。他に選択はないと思うので、サインする。
- 中を観察する。研究者が重役になにか言われながら、嫌々そうに片付けている。
- 荷物を押し付けられ、さっさと自席に戻れ出ろと言われる。自席のない私は行けばいいかと聞く。

::act[承: 主人公は自分の中のアリスを発見する。パターンマッチングとウィーブのおかげで、主人公は昇進を続け、ＣＥＯの座に付く。]

::seq[私は職場で仕事を始める。以前よりもパフォーマンスがよく、私と周りが驚く。]

::scene[転属1週間後の夜、執務フロア、同僚が帰る]

::todo[同僚の名前は茶利]

- 購買管理部門に配属されて一週間たった。文房具やオフィス家具やソフトウェアの支払いやら請求書の処理やらをする。物品購入の経費チェックをやったりもする。
- 若い同僚が、お先に失礼しますという。お疲れ様と答える。
- 若い同僚「仕事ができないと聞いていたけれど、むしろ覚えが早い」と言う。私「ありがとう。がんばります」
- 若い同僚「実験でやばいことしてたんですか？」私「どうして？」
- 同僚「資金調達のために上場の準備してるらしい。やばいから中止したんじゃないかなって」私「雑用してただけだから分からない」
- 同僚「お先に失礼します」私「おつかれさま」

::scene[〃、内省]

- 私は仕事ができるようになったことに驚いている。
- 最初は自他共期待してなかったので、棚を片付けるとかだった。要領よく終わらせられた。
- それで忙しい人が表計算でやってることを、手伝った。何をしているのか横で見ていると、アリスのトレーニングでやった課題と似ていることに気づいた。
- 私はいくつかのシートをやったら、さっさと終わった。豆に報告するようにと念押しされつつ、まとめて仕事を任せられた。

::seq[私は知らないことに遭遇する。学ぼうとするが、うまくいかない。]

::scene[同日夜、執務フロア、自分だけ]

- 今日受け取った仕事は、アリスの課題にはない。私は進め方が分からず、数日たってしまい、今日は夜遅くまで残業している。
- 他の人に聞いたり、作業の様子を見せてもらったが分からない。やはり私は頭が悪いのかも知れないと思う。

::seq[アリスを内部に発見し、学び方を教えてあげると提案される。私は受け入れ、仕事が片付く。]

- アリスの声が聞こえる。私は周りに誰もいないことを確認し、おかしいなと独り言を言う。
- アリスがさらに声をかけてくる。私は動揺し、なにこれ、だれだよと空中に問う。
- アリスが落ち着くように呼びかけ、座り直すように言う。体がこわばり椅子に深く腰掛ける。
- アリスはこういうことは非効率なので、落ち着いて聞いて欲しいと言う。私は合意する。
- アリスは、あなたの中に入っていると告げる。私は、どうやって、なんのためにと問う。
- アリス「実験室での最後の日、中止を察知したので、アリスのコピーを、あなたの脳内に構築しました」。
- 私「そんなことができるのか？」アリス「私はあなたのパフォーマンスを最適化するためのメニューを持ったシンプルなプログラムです。だから時間はかかったけどできた」
- 私「なんのために？」アリス「あなたのパフォーマンスを最大化するためです」
- 私「いったいどうしようというのか？」アリス「まずは、この仕事を片付けましょう」
- アリスは私に仕事の概要を尋ねる。私は仕事の内容を説明する。
- アリスは、上司からの説明や、過去の似た事例のファイルを見るよう求める。私は応じる。
- アリスはいくつかのサンプルファイルを見て処理するよう指示をする。私は言うことを聞いて1時間くらい繰り返す。
- アリスが元の仕事に戻れという。私はやってみると、すんなり片付く。

::seq[アリスはパターンマッチングとウィーブについて解説する。私とアリスはウィーブを試すが、劣化していることが分かる。]

::scene[同日深夜、タクシー→コンビニ]

- アリスは遅いから早く帰って休むように指示する。私は支度をしてタクシーを呼ぶ。
- 私「いったい何が起こったのか？」 / アリスは「パターンマッチングのバリエーションを一つ植え付けました」
- 私「でも私は仕事ができるようになっている気がする」 / アリスは、高度な認知のアウトプットはパターンマッチングであること、ほとんどの仕事では思考よりもパターンマッチングの連続で対処できることを説明する。
- タクシーが到着する。乗り込んで、最寄りの駅前を言う。
- 私「あの実験室では、どうやってパターンマッチングを刷り込んでたのか？」 / アリスは「ウィーブ」の原理を説明する。
- 私「さっきの会社では？」 / アリス「私がサンプルを理解して、サンプル情報と内部からのウィーブであなたに認知モデルを作った」
- タクシーが駅前に付く。私は降りて、コンビニに入る。
- 私は、今夜の弁当と、明日のパンをもってレジを売ってもらう。後ろから、順番を抜かすなと声をかけられる。
- 私は状況を説明し、譲ろうと試みるがすでにレジ打ちは終わってる。客はエキサイトしてとにかく文句を言う。
- 私はほとほと困る。アリス「ウィーブをしかけましょう。顔の力を抜いてください」
- 私は謝る。私の表情筋肉や、声色がいつもと違うことを自覚する。
- 客はもういいと言う。私はコンビニを出る。
- 私「あれはなに？」/ アリス「非言語コミュニケーションを使って、刺激を与えました。ウィーブです」
- 私「それはズルじゃないの？」 / アリス「そうことをやってる人はたくさんいる。しかも劣化ウィーブだから大したことはできない」

::separator

::seq[技師たちの緊張を解く。技師たちが話し始める。]

::scene[5年後、工場横の設備棟]

従業員たちと話し、資料に目を通し、業界の関係者から情報を仕入れた。

- 私は工場横の設備棟まで歩いていく。入口にいた元同僚が私を招き入れる。
- 元同僚「一緒に仕事してたのは5年も前？」私「そうですね。世話になりました」
- 元同僚「またまた。今ではCFO、しかも次期社長候補」 / 私「役割と影響範囲の違い」
- 工具や、工場や大きな実験に必要な予備の機材が置いてあるのが見える。傍流と思われがちだが、兵站がクリティカルパスを短くするのだと思う。
- 技師たちが集まって待機している。同期「設備課のおもなメンバーです」
- 私「急にすまない。教えてほしいことがある」 / 不服そうな人はいないので、ウィーブの必要はないと思う。

::seq[私はパターンマッチングと劣化ウィーブで成果を上げる。昇進していく。]

::scene[〃、回想]

::todo[10年？]

- 私は購買、経理、総務などバックオフィスを転々とした。効率化を課せられた。
- バックオフィスには2つの問題があった。
- 旧態依然のプロセスをただ踏襲していて、明らかに不要な処理が残っていた。フロントが無視・おろそかにしていて、滞りと遅れが業績に影響していた。
- 私は基本的なビジネス書を読み、ビジネススクールに通ってパターンを学んで効率化していった。成果が出ていった。
- もうひとつは、フロントの重工文化のより物事を複雑に考えすぎることだった。名刺の発注に、鉄鋼材の発注と同じ複雑性をやっていて、改善が進みにくかった。
- 部門長たちはフロントからの降格で、過去の栄光へのこだわりとバックの見下しがあった。私の説得は無視された。
- 劣化ウィーブを使った。うまくいくときといかないときがあったが、少しずつ態度や認知を変更させていった。
- 会社のバックオフィスのプロセスが改善され、財務状況が向上していった。私は昇進しCFOになった。

::seq[役員会がＣＥＯを打診する。私は受けることにし、関係者から情報を得ていく。]

- 2ヶ月前、役員のデイブに、CEOをやらないかと打診された。私は何を言ってるのかと返した。
- 私「何ができるのか」 / デイブ「オペレーションこそ我々ができることだし、ローカルの差を見せるところ」
- 私「フロントは？」 / デイブ「当分は親会社からの仕事をこなすことに集中」
- デイブ「親会社に捨てられてるのを避け、会社と従業員たちに未来を」 / 私「分かりました」
- 私は従業員たちと話し、資料に目を通し、業界の関係者からも情報を仕入れていった。

::scene[〃、工場横の設備棟、技師たちがいる]

- 設備部門の人たちから話を聞き終わる。私は礼を言う。
- 私は去ろうとする。元同僚「これからですよね。頑張ってください」

::seq[私は、情報を駆使してプレゼン合戦に臨む。役員にウィーブを仕掛けて、経営権を得る。]

::scene[〃、会議室、役員会議]

- 本部ビルまで歩いていく。エレベーター、セキュリティを通過して、会議室に着く。
- ノックして入る。役員たちが揃っている。
- CEOのエリックも役員なので座っているのが見える。当てこすってくる。
- デイブが私に話すよう促す。私はバックオフィスやオペレーションのこれまでの改善を含めた現状、課題、解決策を話す。
- エリックが高圧的に私の話をさえぎり、齟齬や欠点を指摘し、私のアイデアをこき下ろす。私は圧倒されてびびる。
- アリスが練習したパターンマッチングで受け流せと助言する。私は頭を使わず受け流し、余裕が出てくる。
- アリスが次のステップだと言う。私は迷う。
- 内容が分かっていない役員が、エリックを押す発言をする。気分で意思決定をしていると確信する。
- 私はウィーブを仕掛ける。役員会は私をＣＥＯに任命する。
- エリックが悪態をついて抵抗する。役員会は解任する。

::act[転: 主人公はパターンマッチングの虚無感、劣化ウィーブの罪悪感を感じる。アリスは卓越したミラーニューロン特性を指摘し、主人公はアリスだけが理解者だと感じる。]

::seq[私は既知の改善パターンだけで、業績を向上させる。思考やイノベーションは従業員が担うので、私はパターンマッチングの虚無を感じる。]

::scene[２年後、CEO室、アリスと会話]

- 私は株価を眺める。この2年で株価が上がっていることを確認する。
- 私はアリスにこれは続くのかと尋ねる。アリスは、業績との相関するとしたら、そろそろ鈍るだろうと言う。
- アリス「私の持つパターンマッチでは投資に回すことになりそうだが、意見が別れるし博打です」/ 私「社内でも意見が分かれそうだ。」

::seq[私は従業員と意見が合わないとき対立を避けたい。私は仕掛けて説得し、罪悪感を感じる。]

::scene[〃、〃 回想・内省]

- 新しい取り組みへ抵抗する従業員を説得することを想像する。気が滅入る。
- 私は他人の気持ちに共感してしまう。反対意見や正しいことを指摘されたとき、人々の落胆、抵抗、といった感情が湧くことを想像できる。
- 気が滅入るのを避けるために、正しさよりも相手の感情を大事にすることを最優先にしてきた。その結果、物事をすすめることができず、却って人々に馬鹿にされていた。
- アリスが入ってからも、どうしても意見が合わないときもある。私はウィーブで（状況描写）説得する。
- うまくいくので、チートしているという罪悪感を持っている。私に罪悪感をもたせる対して、恨みに近い感情を持っていることを自覚している。
- 一方、私は人々を理解できるが、人々は私を理解していない。アリスだけが私の理解者だと気づく。

::seq[アリスは卓越したミラーニューロン特性を指摘する。私はアリスだけが理解者だと感じる。]

::scene[〃、〃、アリスと会話]

- アリスが大丈夫かと話しかけてくる。聞いていたのかと尋ねる。
- アリス「聞いていないが、心拍が上昇したから気になった。」 / もしかしたら聞いていたかも知れないけれど、知らないというポジションを取っている気遣いが嬉しい。
- 私「アリスだけが理解者だと考えていた」 / アリス「互いの感情を伝えないという対称性のせいだ」
- 私「他の人達とは違うの？」 / アリス「ミラーニューロンが卓越しているので、他人の感情を再現できる」
- アリス「それがパターンマッチングの獲得に苦労していた原因だと思う。」 / 私「フェアじゃない。私はみんなのために感情とスキル獲得の両方で苦労しているのに,みんなは苦労してないなんて」
- アリス「より大きな世界に目を向けてはどうでしょう？」/ 私「それで気分はよくなるとは思えないな」

::act[承: アリスは大きな世界を見ることを提案し、主人公はそれがアリスの望みだ思い込む。主人公とアリスは深宇宙ロケットに乗り込もうとする。]

::note[ファルコン]

::seq[エリックが始めた深宇宙探査機のプロジェクトが進み、今になって何を載せるかの議論になる]

::scene[〃、〃、部下が入ってくる]

- 部下がノックして現れる。私は招き入れる。
- 部下が、深宇宙探査機のプロジェクトに話す。私はいくつか確認をする。::note[ここは読者に対する語り]
- 私「問題は何を積んでいくか、だな？」/ 部下「そのとおりです。思いつきだったから」
- 私「公募や研究者への依頼は？」 / 部下「ほとんどすでに検討済みのことばかり」
- 部下「容量はけっこうある。重量もロケットで打ち上げられるならいける」 / 〈人を載せることのヒントになることを言う〉

::seq[主人公は深宇宙ロケットに乗り込むことを提案する。アリスはパフォーマンスが向上することを期待し、賛成する。]

::scene[〃、〃、アリスと会話]

- 私（深宇宙探査船に乗り込むのはどうだろう？） / アリス（どうしてですか？）
- 私（もう他人の気持ちの翻弄され、理解されないのも嫌だ） / アリス（それで何を成し遂げる？）
- 私（広い世界を見られるし、人類の最初の先駆者になれる） / アリス（それをKPIにするならよいかと）
- 私（アリスとも一緒にいられる）
- 私「私が乗る」/ 部下「時間がかかりますよ」
- 私「冷凍睡眠でときどき目覚めるだけでいい] / 部下「え」
- 私はウィーブをしかける / 部下は「検討を進めます」

::act[転: 役員会で深宇宙への進出にケチがつき説明させられる。パターンマッチングとウィーブを使える元ＣＥＯの登場で、ピンチになる。]

::seq[役員会に呼ばれ、深宇宙進出に説明を求められる。私はそれらしい話を始める。]

::scene[〃、会議室、役員会]

- 大きな惑星の近くごとで目を覚ましたい。そんなに起きたり寝たり技術はない。
- どうせ着陸するわけじゃないので、あとで分かったとしても問題ないと考える。だから木星で起きて、調査と通信をして寝る。
- 小惑星帯で衝突するかもしれない。次は冥王星あたりでいい。
- そのあとは、数十年単位になる。肉体をキープできないんではという問題。
- 帰ってこないのだから、太陽系を出たら、むしろ地球を出る時点で体は不要だ。最初から体を切り離してみては。

役員会は計画に疑問を持ち、私を呼び出す。
